## Provision drift interface


The build cost scales roughly as:

$$ T(n) = \sum_{i=1}^{n} \frac{c_i}{\log(1 + d_i)} + O(n \log n) $$

where inline $\alpha = \frac{p}{q}$ bounds the drift tolerance.
